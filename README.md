# DDoS Testing Tool

## Overview
This tool is designed to simulate DDoS (Distributed Denial of Service) attacks on specified targets for testing purposes.

## Installation
1. Clone this repository: `git clone https://github.com/yourusername/ddos-testing-tool.git`
2. Navigate to the project directory: `cd ddos-testing-tool`
3. Install dependencies: `pip install -r requirements.txt`

## Usage
Run the tool using the command:
