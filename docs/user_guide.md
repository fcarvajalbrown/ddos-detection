# User Guide

## Overview
This document provides instructions on how to use the DDoS Testing Tool for simulating DDoS attacks.

## Installation
1. Clone this repository: `git clone https://github.com/fcarvajalbrown/ddos-detection.git`
2. Navigate to the project directory: `cd ddos-detection`
3. Install dependencies: `pip install -r requirements.txt`

## Usage
Run the tool using the command:
```bash
python src/main.py
